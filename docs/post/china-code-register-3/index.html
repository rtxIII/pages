软著审核准备-代码文档

代码文档注意事项：

  • 不要有空白行
  • 不要有注释行和注释块，代码结尾处不影响，但是最好也删掉（主要是注释和空白行不能算源代码行数）
  • 建议粘帖后手动整理一下
  • 建议写满 60 页（前 30 页，后 30 页），理论上代码量不足 60 页，全部提交也没问题
  • 字体使用宋体、字号 10、行距 1
